# Arcane Arena

Arcane Arena is a browser-based top-down arena combat prototype built with plain JavaScript ES modules and a single `canvas`.

## Run

Open [index.html](C:/Users/Clay/OneDrive/Documents/ChatGPTProjects/arcane-arena/index.html) in a browser, or serve the project root with any static file server.

## Controls

- `WASD`: move
- Mouse: aim
- Left click: cast equipped primary spell
- `R`: cast equipped secondary spell
- `F` / `T` / `G`: cast unlocked reserve spell
- `Space`: dash
- `P`: pause during arena combat

## Scope

This project follows [SPRINT_PLAN.md](C:/Users/Clay/OneDrive/Documents/ChatGPTProjects/arcane-arena/SPRINT_PLAN.md). The current MVP target is a five-encounter arena run with a menu-based hub, three spells, three enemy archetypes, one boss, and lightweight progression.
